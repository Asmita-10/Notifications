# Notification Dashboard

A modern, minimalist web interface showcasing smooth UI micro-interactions, clean pastel styling, and organized state management for a real-time notification center.

## Features
- **Interactive Triggers:** "Show Notifications" button triggers a sequential, staggered entrance animation.
- **Dynamic Categories:** Categorized status updates including system logs, incoming emails, upcoming calendar events, and direct messages.
- **Premium UI:** Designed with a soft pastel color palette, rounded borders, subtle dropshadows, and a clean bento-grid layout background.

## Tech Stack
- HTML5
- CSS3 (Flexbox/Grid, Keyframes, Transitions)
-  JavaScript (DOM Manipulation)
